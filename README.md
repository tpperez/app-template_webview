## Quick Start

### Prerequisites

- nvm
- node 22.15.1

### Setup

```bash
# install and use node version from .nvmrc
nvm install
nvm use

# install dependencies
npm install

# start development server
npm run dev

# access application
open http://localhost:3000
```
